# Networking basics #0

This project introduces the fundamental concepts of computer networking.
You will learn how devices communicate, how networks are structured, and how to use essential network tools.

## 📚 Learning Objectives

By the end of this project, you should be able to explain, without using Google:

### 🔹 OSI Model
- What the OSI model is
- How many layers it has
- How it is organized

### 🔹 LAN, WAN, Internet
- What a LAN is, its usage and geographical size
- What a WAN is, its usage and geographical size
- What the Internet is

### 🔹 IP & Addresses
- What an IP address is
- The two types of IP addresses (private/public)
- What localhost is
- What a subnet is
- Why IPv6 was created

### 🔹 TCP & UDP
- The two main transport protocols used over IP
- The main difference between TCP and UDP
- What a port is
- Port numbers to memorize: **22 (SSH), 80 (HTTP), 443 (HTTPS)**

### 🔹 Tools
- What tool/protocol is used to check if a device is connected to a network (ping/ICMP)
- How to use `netstat` and `ping`
